CSS Pseudo Selectors Demonstration

1. :hover Pseudo-class

The :hover selector applies styles when you move your mouse over an element. It's perfect for creating interactive effects and improving user experience.
Hover Over Me!
.hover-demo:hover { background: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%); transform: scale(1.05); box-shadow: 0 8px 20px rgba(0,0,0,0.3); }

2. :link Pseudo-class

The :link selector targets hyperlinks that have not been visited yet. It only works on elements with an href attribute.
Unvisited Link 1 Unvisited Link 2 Unvisited Link 3
a:link { background: #667eea; color: white; border: 2px solid #667eea; }

3. :active Pseudo-class

The :active selector applies styles during the moment you click on an element. Try clicking and holding on the link below!
Click and Hold This Link
a:active { background: #f39c12; color: white; transform: scale(0.95); }

4. :first-child Pseudo-class

The :first-child selector targets the first element among a group of sibling elements. Notice how the first item in the list below has special styling.
  • First Item - I'm Special!
  • Second Item - Regular styling
  • Third Item - Regular styling
  • Fourth Item - Regular styling
  • Fifth Item - Regular styling
.list-demo li:first-child { background: linear-gradient(135deg, #fa709a 0%, #fee140 100%); color: white; font-weight: bold; font-size: 1.1em; }

5. :visited Pseudo-class

The :visited selector changes the appearance of links that have been clicked. Click the links above to see them change color!

a:visited { background: #95a5a6; color: white; border: 2px solid #95a5a6; }

6. ::first-line Pseudo-element

The ::first-line selector applies styles only to the first line of text in an element. Notice the double colon (::) for pseudo-elements versus single colon (:) for pseudo-classes.

This is the first line of the paragraph and it has special styling that makes it stand out from the rest of the text. As you can see, only the first line is styled differently with larger, bold, red text in uppercase. The remaining lines maintain normal formatting so you can clearly see the difference between the first line and the rest of the paragraph content.

.text-demo p::first-line { color: #e74c3c; font-weight: bold; font-size: 1.3em; text-transform: uppercase; }

Summary of CSS Pseudo Selectors

  • :hover - Applies when mouse hovers over element
  • :link - Targets unvisited hyperlinks
  • :active - Applies when element is being clicked
  • :first-child - Selects the first child element
  • :visited - Targets links that have been clicked
  • ::first-line - Styles only the first line of text

Note: Pseudo-classes use a single colon (:) while pseudo-elements use double colons (::)
